# Manufacturing OEE Dashboard

## Purpose
A comprehensive dashboard for visualizing 24/7 manufacturing production data, focusing on Overall Equipment Effectiveness (OEE), downtime, and quality metrics based on a provided database schema.

## Key Features
- **OEE Monitoring**: Real-time tracking of Overall Equipment Effectiveness with visual gauges and progress indicators
- **Production Analytics**: Comprehensive production logging, defect tracking, and quality metrics visualization
- **Maintenance Management**: Preventive maintenance scheduling, work order management, and maintenance calendar
- **Shop Floor Visualization**: Interactive shop floor layout with machine status monitoring
- **Inventory Management**: Spare parts inventory tracking and consumable request management
- **Multi-language Support**: Internationalization with language switching capabilities
- **AI-Powered Analysis**: Integration with Google Gemini AI for intelligent data analysis and insights
- **Interactive Charts**: Rich data visualization using Recharts library including heatmaps, Pareto charts, and trend analysis

## Target Users
- Manufacturing Operations Managers
- Production Supervisors
- Maintenance Teams
- Quality Control Personnel
- Plant Engineers

## Use Cases
- Monitor real-time production efficiency and OEE metrics
- Track and analyze equipment downtime patterns
- Schedule and manage preventive maintenance activities
- Analyze defect trends and quality issues
- Manage spare parts inventory and requests
- Generate production reports and performance benchmarks
- Visualize shop floor status and machine health
